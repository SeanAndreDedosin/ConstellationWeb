What is HTML?

  • HTML stands for Hyper Text Markup Language.
  • HTML is the markup language displayed on the page.
  • HTML is used for web development, internet navigation and web documentation.

    • The Five Main Essential Tags!




    Adding Images

  • To add images you need to use the image tag.
  • the image tag has no closing tag.
  • alt is the alternative text for the image
  • src is the filepath.



    • Creating Tables

  • HTML tables allow data to be displayed in columns and rows.
  • td displays table data.
  • th displays table stands for table header.
  • tr displays table rows.



    • Using Class and Id Attributes

  • HTML uses class attributes.
  • HTML elements can use the same class.
  • Class can be used in stylesheet and javascript.
  • It manipulates elements with a specific class name.
  • Class can be used on any HTML element.
  • Class name is case sensitive.
  • Javascript access elements with specific class name with getElementByClassName()



  • HTML id used to specify a unique id.
  • Id attribute must be unique within the html document.
  • Id can be used in stylesheet and javascript.
  • Id can be used on any HTML element.
  • Id name is case sensitive.
  • Javascript access elements with specific class name with getElementById()
